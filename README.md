# 麻雀一番街切号器

一个适用于 Windows 的本地桌面工具，用于保存、管理并切换麻雀一番街账号登录状态。

本工具通过读取和写回 Windows 注册表中的游戏登录数据，在同一台设备上实现多个账号的本地切换。

## 适用范围

适用于以下场景：

- 在一台 Windows 电脑上使用多个麻雀一番街账号
- 希望保存常用账号，减少重复登录操作
- 需要在 Steam 版和官网版之间选择启动方式

## 系统要求

- Windows 10 或 Windows 11
- 已安装麻雀一番街客户端
- 已至少成功登录过一次游戏

如使用源码运行：

- Python 3.11+

说明：

- 本工具依赖 Windows 注册表，仅适用于 Windows。
- 登录数据来自注册表 [`SOFTWARE\\HappyWoods\\RiichiCity`](services/registry_service.py:9)。
- 如果游戏从未成功登录，工具可能无法读取到可导入的账号数据。

## 核心功能

- 导入当前已登录账号
- 保存多个账号的本地登录快照
- 编辑账号备注
- 删除单个账号
- 批量删除账号
- 调整账号列表顺序
- 切换账号前自动关闭游戏进程
- 切换账号后自动启动游戏
- 支持 Steam 启动
- 支持选择官网版游戏可执行文件启动
- 查看环境信息
- 记录运行日志

## 快速开始

### 1. 启动游戏并登录账号

首次使用前，请先正常启动麻雀一番街并登录一个账号，确保系统中已经生成对应的登录数据。

### 2. 启动工具

#### 方式 A：直接下载发布版

1. 打开 GitHub Releases
2. 下载 `MahjongSwitcher-vX.Y.Z-windows-x64-portable.zip`
3. 解压到任意本地目录
4. 双击 `MahjongSwitcher.exe`

注意：

- 不要在压缩包内直接运行
- 用户数据与日志位于 `%LOCALAPPDATA%\\MahjongSwitcher`

#### 方式 B：使用源码运行

在项目根目录执行：

```bash
python launcher.py
```

### 3. 配置启动方式

打开工具后，在“运行配置”区域选择启动方式：

- Steam 版
- 官网版

如使用官网版：

1. 切换到“官网版”
2. 点击“选择游戏位置”
3. 选择正确的游戏 `.exe` 文件

相关设置会保存到 [`data/settings.json`](data/settings.json)。

### 4. 导入当前账号

点击“导入当前登录账号”，将当前已登录账号保存到账号列表。

### 5. 切换账号

在账号列表中找到目标账号，点击“切换账号”。

工具会先尝试关闭当前游戏进程，再写入目标账号数据，并按当前设置重新启动游戏。

## 使用说明

### 导入账号

点击“导入当前登录账号”后：

- 工具会读取当前登录账号的数据
- 导入成功后，账号会显示在列表中
- 默认备注为“账号1”“账号2”等
- 账号数据会保存到 [`data/accounts.json`](data/accounts.json)

如果未读取到账号数据，通常表示当前没有可用的登录数据。

### 管理账号

账号列表支持以下操作：

- 切换账号
- 编辑备注
- 删除账号
- 批量删除账号
- 调整显示顺序

批量删除时，可先进入“批量删除账号”模式，再勾选目标账号执行删除。

### 调整账号顺序

可以直接拖拽账号项调整显示顺序。

顺序会保存到 [`data/account_order.json`](data/account_order.json)，下次启动时仍会保持。

### 查看环境信息

点击“查看环境信息”可查看以下内容：

- 当前启动方式
- 官网版路径
- 数据目录
- 游戏进程名
- 账号数量
- 日志文件位置
- 环境检测结果

环境检测内容包括：

- 当前系统是否为 Windows
- 游戏进程名是否已配置
- 注册表是否可访问
- 官网版模式下游戏路径是否有效

### 启动游戏

点击“启动游戏”可按当前配置直接启动游戏。

- Steam 模式下会通过 Steam 协议启动
- 官网版模式下会启动所选的本地 `.exe` 文件

如果官网版路径无效，工具会提示重新配置。

## 数据与日志

运行过程中会使用以下本地文件：

- `%LOCALAPPDATA%\\MahjongSwitcher\\data\\settings.json`：启动方式、官网版路径、进程名等设置
- `%LOCALAPPDATA%\\MahjongSwitcher\\data\\accounts.json`：已保存账号的数据和备注
- `%LOCALAPPDATA%\\MahjongSwitcher\\data\\account_order.json`：账号列表顺序
- `%LOCALAPPDATA%\\MahjongSwitcher\\logs\\app.log`：运行日志

程序运行时会自动创建 `%LOCALAPPDATA%\\MahjongSwitcher\\data` 和 `%LOCALAPPDATA%\\MahjongSwitcher\\logs` 目录。

## 限制与注意事项

- 仅适用于 Windows。
- 依赖本机注册表中的游戏登录数据。
- 切换账号前会尝试关闭游戏进程。
- 如果游戏进程在 15 秒内未退出，切换会中止，并提示手动关闭后重试。
- 使用官网版启动时，必须选择有效的游戏 `.exe` 文件。
- 删除账号会删除本地保存的账号记录，无法在工具内恢复。
- 工具保存的是本地登录快照和备注信息，不包含账号密码管理功能。

默认游戏进程名为 [`Mahjong-JP.exe`](models/settings.py:11)。

## 常见问题

### 无法导入当前账号

常见原因：

- 游戏尚未成功登录
- 当前系统中不存在对应注册表项
- 当前不是 Windows 环境
- 账号数据读取失败

处理建议：

1. 先手动启动并登录游戏
2. 确认已进入账号可用状态
3. 返回工具重新执行导入

### 切换账号失败，提示官网版路径无效

表示当前使用的是官网版启动方式，但本地游戏路径无效或未配置。

处理方法：

1. 打开工具
2. 切换到“官网版”
3. 重新选择正确的游戏 `.exe` 文件

### 切换账号失败，提示游戏未在 15 秒内退出

表示工具未能在限定时间内关闭游戏进程。

处理方法：

1. 手动关闭游戏
2. 确认游戏进程已经完全退出
3. 重新执行切换

### 账号顺序没有按预期显示

账号顺序保存在 `%LOCALAPPDATA%\\MahjongSwitcher\\data\\account_order.json`。如该文件内容异常，顺序可能回退或被重新整理。

### 需要排查问题时看哪里

优先查看日志文件 `%LOCALAPPDATA%\\MahjongSwitcher\\logs\\app.log`。

日志中会记录设置保存、账号导入、账号删除、启动游戏、切换账号和环境检测等运行信息。
